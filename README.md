# FAF Skills Site

Static website for skills.faf.one - Browse and discover 37 Claude Code skills.

## About

This is the web portal for browsing FAF skills available for Claude Code. It follows the same static HTML pattern as foundation.faf.one.

## Features

- 🔍 Search 37 skills across 5 categories
- 📋 One-click copy skill commands
- 🎨 FAF ecosystem design consistency
- 📱 Responsive design
- ⚡ Zero dependencies, pure HTML/CSS/JS

## Skills Categories

- **FAF** - Core FAF operations (13 skills)
- **Development** - Git, code review, maintenance (5 skills)
- **MCP** - Model Context Protocol tools (3 skills)
- **Integration** - n8n and other integrations (2 skills)
- **Documentation** - PRD, architecture, requirements (3 skills)
- **Publishing** - npm, PyPI, crates.io (4 skills)
- **Utility** - Helper tools (1 skill)

## Local Development

```bash
# Clone the repository
git clone https://github.com/Wolfe-Jam/faf-skills-site
cd faf-skills-site

# Open in browser (no build needed)
open index.html
```

## Deployment

This site is deployed to skills.faf.one via Vercel.

```bash
# Deploy with Vercel
vercel --prod
```
